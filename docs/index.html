Metodología

La base se homologa a partir de BASE_FAHU.xlsx, normalizando nombres de columnas y tipos de datos antes del análisis.
Las horas del claustro se calculan por unidad del profesor, no por la unidad donde se dicta el curso. Los sellos se deduplican por profesor y sección.
Las autoridades suman horas a la unidad, pero no entran al divisor del promedio del claustro. Las secciones sin profesor y las inconsistencias TELS se reportan aparte.
La síntesis y los comparativos se generan a partir del mismo conjunto limpio, para mantener consistencia entre informes por unidad, comparativos históricos y salidas LaTeX.
